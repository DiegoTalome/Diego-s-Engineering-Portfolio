VHDL Computer Design

This project uses VHDL to design a simple computer architecture implemented on an FPGA board.

Register File

The register file consists of multiple 8-bit registers with write and dual-read capability. It uses a write-enable control line, and multiplexers select which register's data is placed on the output buses. Address inputs determine which register is read or written to.

ALU (Arithmetic Logic Unit)

The ALU supports operations such as ADD, SUB, AND, OR, and XOR on 8-bit operands. Flags such as Zero and Carry are calculated as outputs. It forms the computational heart of the processor and operates based on a 3-bit opcode input.

Three Stage Amplifier

A multi-stage analog amplifier built to demonstrate gain, bandwidth, and signal fidelity across each stage.

Design Overview

This amplifier consists of three sequential gain stages: a common-emitter amplifier for initial voltage gain, a second common-emitter stage for increased amplification, and an emitter follower (buffer) stage to provide current drive capability with minimal output impedance. Each stage is carefully biased to operate in the active region, and coupling capacitors are used to block DC while allowing signal transfer.

Performance

The amplifier was tested with a 1 kHz sinusoidal input signal. The first two stages provide the bulk of the gain, while the final stage maintains signal integrity when driving a load. Total gain achieved was approximately 7.1 V/V, with stable operation and minimal distortion across the frequency range of interest.
