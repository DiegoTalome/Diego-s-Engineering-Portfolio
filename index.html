Readme

Description: This C++ game uses stacks to model rooms, allowing navigation between them, and incorporates basic object manipulation using push and pop operations.

Stack Functionality:
The game uses the STL stack container to manage room tiles. Each room is represented as a stack where tiles are added (via push) and removed (via pop), mimicking the natural behavior of a stack data structure.

  • push: Adds a tile to the room stack, representing elements such as doors (/) or treasure (*).
  • pop: Removes the top tile to process room data dynamically.

Pointers:
Pointers are used in this game to navigate through the current room stack. Instead of managing memory directly, STL handles memory allocation internally for stack operations.
While pointers do not directly manipulate addresses in this game, they track the player's position in the room stack, avoiding the complexities of direct memory address management.

Memory Implementation:
If a manual memory stack were implemented instead of using #include <stack>, you would need to:

  1. Allocate memory for each stack element dynamically.
  2. Manage memory explicitly using new and delete for each push and pop operation.
  3. Write additional logic to ensure stack overflow and underflow are managed safely.
This approach would increase complexity but allow finer control over memory usage.

#include <iostream>
#include <vector>
#include <stack>
#include <string>
#include <algorithm>

using namespace std;

typedef stack<char> Room;

vector<Room> rooms;  
int currentRoom = 0; 
int currentPointer = 0;
bool treasureGrabbed = false; 
int treasureRoom = 3; 

void initializeRooms() {
    Room room1, room2, room3, room4;
    room1.push('-'); room1.push('/'); room1.push('-'); room1.push('-'); room1.push('-'); room1.push('-'); room1.push('-'); 
    room2.push('-'); room2.push('-'); room2.push('/'); room2.push('-'); room2.push('-');
    room3.push('-'); room3.push('/'); room3.push('-'); room3.push('-'); room3.push('-');
    room4.push('-'); room4.push('-'); room4.push('*'); room4.push('-'); room4.push('-');
    rooms.push_back(room1);
    rooms.push_back(room2);
    rooms.push_back(room3);
    rooms.push_back(room4);
}
