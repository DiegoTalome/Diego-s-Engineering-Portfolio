Register File

A register file is a small, fast memory unit within a CPU that stores temporary data needed during instruction execution. Unlike RAM, the register file offers quick access to values used in arithmetic, logic, and data movement operations.

Each register is built with flip-flops and stores a fixed-width word (e.g., 8 bits). Flip-flops are synchronized with a clock and controlled by enable and reset lines.
