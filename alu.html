ALU Design

An Arithmetic Logic Unit (ALU) is a core component of the CPU responsible for executing operations like addition, subtraction, bitwise AND/OR/XOR, and comparisons. It uses input from the register file and returns results back into memory or registers.

This ALU design project is currently under development. Diagrams, waveform simulations, and implementation details will be added soon.

Project for UAF EE 443 LAB with Dr. Raskovic.
